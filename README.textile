h1. Data

This repository will contain data sets based on or in addition to datasets available from the UCD, CLDR and various data sets availabe from SIL Interational.

The purpose is to povide access to the data which can be used to support font and keyboard layout design for language enablement projects.

h2. External data sets

Core data used for developing langauge solutions include:

* Unicode Character Database: "Text":http://www.unicode.org/Public/UCD/latest/ucd/ , "XML":http://www.unicode.org/Public/UCD/latest/ucdxml/
* Unicode Code Charts: "index":http://www.unicode.org/charts/ , "all":http://www.unicode.org/Public/UCD/latest/charts/CodeCharts.pdf
* Unicode Common Locale Data Repository: "CLDR data":http://cldr.unicode.org/index/downloads
* ISO 639-3: "SQL D/b formated data":http://www.silinternational.net/iso639-3/download.asp
